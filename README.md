# Spark Cards App

## 🚀 What I Built

This is a dating-style card swipe app built using React.
Users can view profiles, like them, and interact with the UI.

## 💡 Features

* Profile cards
* Like functionality ❤️
* Restart button
* Smooth UI interactions

## 🛠️ Technologies Used

* React.js
* JavaScript
* CSS

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/yourusername/spark-cards-kavya.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the app:

```bash
npm start
```
